## Developer-tool installation and testing

Supported platforms: Windows, macOS, and Linux with Node.js 20 or newer.

Fast test path: open https://unwritten-xi.vercel.app.

Local path:

```bash
git clone https://github.com/Z3X-1337/unwritten.git
cd unwritten
npm test
npm run check
npm start
```

Open `http://localhost:3000`. The bundled judge path uses synthetic evidence and in-memory refund/ticket tools; it does not connect to real payment or support systems.
